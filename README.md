Input the number of elements to store in the array :3
Input 3 number of elements in the array :
* element - 0 : 2
* element - 1 : 5
* element - 2 : 7

Expected Output :
The values store into the array are :
2 5 7
The values store into the array in reverse are :
7 5 2
